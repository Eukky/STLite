# STLite

C++ STL的轻量版复现。

- 用于学习STL
  
- 用于学习范型编程

## 模块

- 空间分配器
  - alloc: 空间配置实现相关函数
  - allocate: 对空间配置函数进行封装
  
- 迭代器
  - iterator: 迭代器实现

- 容器
  - vector

- 算法
  - algorithm

- 仿函数

- 配接器

- 其他
  - construct: 构造与析构相关函数
  - uninitialized: 对未初始化空间进行配置的全局函数
  - type_traits: 偏特化型类型萃取器

- 测试
  - vector_test
